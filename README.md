# Melbourne Property Price Predictor

Predicts residential property sale prices using the Melbourne Housing dataset.

## Tools used
Python, XGBoost, scikit-learn, SHAP, pandas, seaborn

## Results
- Best model: XGBoost (R² = 0.85, RMSE = $281,112)
- Key drivers: Distance from CBD, Building Area, Rooms

## How to run
1. Download `melb_data.csv` from Kaggle
2. Install dependencies: `pip install -r requirements.txt`
3. Open `melbourne_property_predictor.ipynb` in Jupyter
